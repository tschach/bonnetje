What is bonnetje?

bonnetje is a free & open source web application for creating a receipt image from an album by your favourite artist or band.

The idea for this was made popular by @albumreceipts some years ago.

With bonnetje you have control over the look and content of the receipt. You can also create a concert receipt and turn it into a memorable fan freebie.

The images created with bonnetje are free to use for any purpose. No need to ask for permission.

How to use it

You can load the examples at the top to learn how it works.

The receipt preview image is updated in real-time with your inputs.

With save data and load data you can save the current state of the form (including the logo image) to your computer or device and restore it later.

All input fields are optional (except for the tracklist). If you leave them empty, they will not appear on the receipt.

Use download image to save the receipt image. You can not save it by right-clicking on the image in the browser or long-touching it on a phone.
